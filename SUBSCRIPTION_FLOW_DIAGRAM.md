## ğŸ¯ Important Objects

### SubscriptionManager
- **Type**: ObservableObject (SwiftUI state management)
- **Lifetime**: Entire app session
- **Purpose**: Single source of truth for subscription state

### PaywallView
- **Type**: SwiftUI View
- **Presentation**: Full screen cover
- **Dismissal**: Automatic when `isSubscribed` becomes true

### Product
- **Type**: StoreKit 2 struct
- **Contains**: Price, description, product ID
- **Source**: App Store or StoreKit configuration

### Transaction
- **Type**: StoreKit 2 struct
- **Contains**: Purchase proof, expiration, offer type
- **Verified**: By Apple's signature
